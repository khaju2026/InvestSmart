# InvestSmart
Aplicação de investimentos com gráficos interativos, indicadores e notícias financeiras.

## 🚀 Funcionalidades
- Dashboard com gráficos de ativos usando Plotly.
- Indicadores técnicos e exportação para Excel.
- Notícias financeiras via RSS (Infomoney por padrão).
- Deploy simples no Railway.
